XITY :

It is a platform which contains the listings for a wide range of businesses across multiple categories.A user can search for listings by locating through pincodes and get connected to the business . Anybody can have a business and that person can host its business on this website

Developed the frontend of the website in angular using HTML , CSS and typescript .
Made use of frameworks such as prime-ng , bootstrap (ng and ngx included) and material UI to make the development process even faster and 100 percent responsive. Used libraries such as apache e-charts and ngx e-charts to display pictorial data.
Implemented map functionalities using google maps and added custom funtionalities.
Developed APIs for all the functionalities in the front end as well as admin panel.
Used mongo DB as database and amazon s3 as well as google firebase for storage.Written complex queries using mongo DB aggregation pipelines.
Used hashing techniques that are industry standard as usual.
Used libraries such as date-fns and moment js to handle dates.
and much more...
