AQUATIK ANGELS :

https://aquatikangels.com/

It is an e-commerce website for purchasing aquatic products like aquariums , sea animals etc , It also consists of a scaping module that consists of taking orders based on custom aquariums.

TECH : Angular 17 , Node JS , Mongo DB , Express JS , Fabric JS , Canvas

  • The website is developed in Angular 17 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • Developed all the features that are required for an ecommerce , added url based filtering and sorting , and other stuffs are already there like cart , order , inventory , user management etc
  • Made use of ngx-pagination to implement server sided pagination wherever required.
  • Reduced load time by converting images to webp and storing on s3 , all handled on the server.
  • Created a module for aquascaping , basically a dynamic customised aquarium that users can order with their requirements using fabric js.

AQUATIK ANGELS ADMIN PANEL / BACKEND:

https://admin.aquatikangels.com/

The portal to handle the contents of the main website , and all other ecommerce features like order management , stock management etc

  • Made dynamic scape materials for aquascaping contents
  • Integrated razorpay payment gateway to handle payments
  • Made use of prime ng tables to display tabular data with all the necessary actions as well.
  • For the backend , developed REST APIs using express JS , mongoose etc.
  • Developed optimized queries for fetching server sided data for the tables in admin as well as for the data in the main frontend website.
  • Developed advanced queries using mongodb aggrgation pipelines for fetching complex data.
  • Worked with dates using moment js to ftech data as per current date and time.
  • Used aws s3 to store media files with the help of multer js.
