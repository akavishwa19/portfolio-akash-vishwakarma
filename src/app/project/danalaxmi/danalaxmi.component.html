DANALAXMI MATRIMONY GROUP:

https://danalaxmimatrimonygroup.com/

It is a platform which offers matrimonial services based on caste system which is active for over 20,000 + users

TECH : Angular 17 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 15 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • Developed features such as liking profiles , viewing contacts , etc
  • Developed complex filters for searching profiles
  • Developed server sided pagination using angular materials pagination component

DANALXMI ADMIN PANEL / BACKEND:

https://admin.danalaxmimatrimonygroup.com/

The admin panel can control the profiles for the main website as well as add testimonials , add plans etc

  • Implemented server sided rendering into datatables.
  • Develpoed admin panel to add users into the portal which then get their user credentials in their mail , the mail is sent via using sendgrids apis. Using these credentials the user can log in into the frontend website and search for matches corresponding to their own caste.
  • Used JWTs for authentication purposes and used bcrypt js for hashing passwords into the database.
  • Used google firebase to store images in multiple formats , generated by using sharp js
  • Used amazon s3 to store the pdfs and images of profiles.
  • Used html-pdf-node for generating pdfs on the server and then storing them as the profiles get edited or saved
  • Implemented pdf download for single profiles as well jpeg downloads using pdf-2-pic
  • pdf-img-convert for converting the pdfs into images and storing them as well on s3
  • Implemented bulk pdf download as well as bulk image downloads using pdf-lib where in case of images , the generated images are zipped and then sent on client as a buffer which is then opened using the package file-saver , and in case of pdfs they are clubbed in a single pdf and that pdf is sent to client as buffer which is opened using file-saver.
  • Developed advanced queries using mongodb aggrgation pipelines for fetching complex data , performing advances filters , sorting etc.
