VEEV BAGS :

https://veevbags.com/

Veev Bags is a premium e-sommerce website for selling bags of various types online , the bags range from Rs 1000 to Rs 50000.

TECH : Angular 18 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 18 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • Developed complete responsive UI from scrath.
  • Used swiper js and slick carousel for sliders in the app.
  • Used ngx skeleton loader for between get calls for a sleek look
