MINI-ECOMMERCE :

https://mini-ecommerce-iota.vercel.app/

Its basically an e-commerce platform with basic features like cart and order management just to demonstrate how an e-commerce functions

TECH : Angular 18 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 18 with bootstrap as the styling library
  • Implemented authentication using jwt and http cookies , with basic ecommerce features such as ( inventory , cart , order ) management and other necessary ecommerce features.
  • Included role based authentication for logging in as a user and the admin to manage products on the website
  • Made use of cloudinary to handle storage of media on the website.
  • Developed all other APIs that are used in the frontend
