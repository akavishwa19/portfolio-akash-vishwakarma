YMCA :

https://ymcaindia.com/

It is the website for YMCA india , containing details about all the projects , services , regions and other details about the institution

TECH : Angular 18 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 16 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • We also had a theme which we installed to adopt componenets directly into the website.
  • Made use of ngx-pagination to implement server sided pagination wherever required.
  • Integrated the APis that serve data from the admin panel

YMCA ADMIN PANEL / BACKEND:

https://admin.ymcaindia.com/

It is the powerhouse for the main webiste of ymca india

  • The admin panel is developed in angular 18 with similar UI libraries in the front end.
  • Integrated and developed APIs for the content in the main panel
  • Used s3 for storage
  • Made use of prime ng tables to display tabular data with all the necessary actions as well.
