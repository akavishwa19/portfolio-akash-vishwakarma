DOC MANAGEMENT TOOL :

It is a tool made for MAAS Global for managing their documents , basically you are creating a book altogether that is represented beautifully when accesses from the main website.

TECH : Angular 17 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 15 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • It also has an admin panel that offers to add documents based on the organisations under the company.
  • Added tinymce for contant block and for addition of images based on position of the page , ng-select is used.
  • Used firebase for storage
