SD COVERINGS :

https://sdcovering.com/

SD Covering is an advance e-commerce platform to buy jewelleries online.

TECH : Angular 17 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 17 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap ,prime ng , ant-design , ng-zoro for styling the componenets.
  • Creted advance filters and sorting with query params in the url that benefit in url sharing while keeping filters on , seo benefits etc.
  • Used signals for handling cart updation
  • We also have the feature of having simple and products with variants
  • Made use of signals for emitting data across components
  • Made use of interceptors for authentication and attaching request headers.
  • Integrated seamless checkout process and cart management etc
  • Used swiper js for sliders

SD COVERING ADMIN PANEL / BACKEND:

https://admin.sdcovering.in/

The admin panel controls the content of the main webiste that is aayra Jewellery

  • Developed dynamic role allocation , that is we can attach permissions to roles , hence restricting admins to access paticular sections of the admin panel based on the roles , thus dynamic roles for role based access
  • Developed admin panel to track orders , manage inventory , manage admins/users , add products with variants , add coupons etc
  • Used JWT , HTTP only cookies , RS-256 for authentication and stored the private and public key in file system that gets created using crypto js and is updated using cron job that runs every 7 days
  • Used url-slug for creating product slugs , category slugs etc and uuid for creating unique ids
  • Used google firebase as well as amazon s3 for storage of multimedia using multer js
  • Used moment js for handling banners based on active dates
  • Developed optimized queries for fetching server sided data for the tables in admin as well as for the data in the main frontend website.
  • Developed advanced queries using mongodb aggrgation pipelines for fetching complex data .
  • Handled cart operations , wishlists , checkout ,orders atc
  • Made use of mongodb transactions , aggregation pipelines etc for handling checkout and stock management
