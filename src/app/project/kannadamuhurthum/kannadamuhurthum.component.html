KANNADA MUHURTHAM :

https://kannadamuhurtham.com/

It is a matrimony website in which users can signup / log in and search for their partners online and get married if they find their choice.

TECH : Angular 18 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 16 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • Made use of ngx-pagination to implement server sided pagination wherever required.
  • Added complex sorting and filtering features

KANNADA MUHURTHAM ADMIN PANEL / BACKEND:

https://admin.kannadamuhurtham.com/

It is the powerhouse for the main webiste

  • The admin panel is developed in angular 18 with similar UI libraries in the front end.
  • It has dynamic role based access implementation based on permissions of menu items in the admin panel
  • The admin panel can control content of all the filters , profiles etc
  • Developed complex logics for matching of profiles etc
  • Developed complex logics for sending interests , shortlisting , viewing contacts , viewing profiles etc
