INSHLT :

https://www.inshlt.org/

INSHLT is an organisation for the doctors of heart and lunf transplantation society of India.

TECH : Angular 15 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 15 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • Used theme and modified it to get the UI
  • Added functionality to add events , news , gallery etc via the admin panel.
  • Handled dates using moment js.
  • Stored media files on google firebase/storage
  • Used interceptors to attach request headers and jwt token.
  • Integrated razorpay payment gateway for creating orders , checkout , capture etc.
