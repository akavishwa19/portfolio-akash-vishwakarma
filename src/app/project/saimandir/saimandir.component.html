SAI MANDIR CHARLOTTE :

https://saimandhircharlotte.org/

Sai Mandir Charlotte is a US based temple website , it gives the users updates about the festivals ,events , pujas etc . It also offers users to donate to the temple online.

TECH : Angular 15 , Node JS , Mongo DB , Express JS

  • The website is developed in Angular 15 with angular material ,bootstrap ,ng-bootstrap and ngx-bootstrap for styling the componenets.
  • Developed the pages that are fed with dynamic content throughout ,eg : events , festivals , activities etc
  • Handled dates using moment js
  • Developed admin dashboard using nebular , that controls the content for the main website.
  • Generated pdfs on the server for invoice after payment using html-pdf-node and pdf-lib
  • Integrated role based authentication for user , admin and superadmin login
  • Used sendgrid to send mail for OTP based login
